Github Pages加速

2021-02-20

本网站是我使用Github Pages+Gridea搭建的静态博客，域名当初买了趁着搞活动41元买了10年，计划长期使用，然而使用过程中发现这套配置唯一的短板就是国内访问速度太慢，因此收藏几个可以加速访问的方法。

加速Github Pages:

  • Github Pages访问太慢？通过Netlify免费加速。
    https://www.shuzhiduo.com/A/LPdovX1O53/

免费，以前用过一段时间，速度还是不错的，但也有一些问题：
一是经常会出现点击登录之后就发生错误，导致无法登录，累觉不爱；其次用了一段时间后打开博客发现网速好慢，感觉效果可能不太行了。（后面我是在家里访问，也许是因为这边网络比较差。）

  • 优雅的使用Vercel加速GitHub Pages
    https://www.iaia.cc/posts/2418998582.html

免费，虽然加速效果有限，但免费还要啥自行车。印象中之前Netlify配置好后速度比Vercel快点。
前几次弄好后发现不但没有加速，反而博客打不开了。最后看了邮件，原来一定要加入另外两条解析记录。

  • 使用国内CDN加速
    https://www.cnblogs.com/geraint/p/13115459.html

国内CDN主要是腾讯云、阿里云等这些国内互联网公司提供的服务。
收费，腾讯云新人会送一些流量但我不想&懒得备案域名，所以没用过。
