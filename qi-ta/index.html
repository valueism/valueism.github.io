github page加速

2021-02-20

本网站是使用github page搭建的静态博客，域名当初买了趁着搞活动41元买了10年，计划长期使用，然而使用过程中发现唯一的短板就是访问速度太慢，因此收藏几个可以加速访问的方法。

加速github page:

  • Github Pages访问太慢？通过Netlify免费加速。
    https://www.shuzhiduo.com/A/LPdovX1O53/
    Netlify经常会出现点击登录之后就发生错误，无法登录，累觉不爱。
    之前（2020年）Netlify的加速效果还是比较可观的，虽然比国内一般网站慢一些，但完全在可以忍受的范围内。最近打开博客发现网速好慢，于是决定换掉它。（这次我是在家里访问，也许并不是加速效果而是家里宽带网络的问题。）

  • 优雅的使用Vercel加速GitHub Pages
    https://www.iaia.cc/posts/2418998582.html
    免费，虽然加速效果有限，但免费还要啥自行车。印象中之前Netlify配置好后速度比Vercel快点。

  • 使用国内CDN（腾讯、阿里云等）加速
    https://www.cnblogs.com/geraint/p/13115459.html
    收费，即使送流量但懒得备案域名，所以没用过
