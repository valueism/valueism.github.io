剑指 Offer 05. 替换空格
请实现一个函数，把字符串 s 中的每个空格替换成"%20"。

示例 1：
输入：s = "We are happy."
输出："We%20are%20happy."

限制：
0 <= s 的长度 <= 10000

实在想不出有什么特殊技巧，用最简单的方法纯手动实现

class Solution {
public:
    string replaceSpace(string s) {
        string str;
        for(auto i:s){
            if(i==' '){
                str+="%20";
                continue;
            }
            str+=i;
        }
        return str;
    }
};
运行效果

如果没有限制，或者看重时间效率，那么上面的代码已经足够了。但是不同的时间/空间需求也可以有不同的解法，如果看重空间效率，那么不应该像上面那样生成新的字符串。

由于有两种不同的解决方案，我们应该 向面试官问清楚 ，让他明确告诉我们他的需求。假设面试官让我们在原来的字符串上进行替换，并且保证输入的 字符串后面有足够多的空余内存。

后面列举了两个例子，第一个是较差的时间复杂度为O(n^2)的解法，即每次替换将空格后面的字符全部向后移动两个位置；
第二种效率较高，时间复杂度为O(n)，同时没有生成新字符串，相比最上面的会更好。
